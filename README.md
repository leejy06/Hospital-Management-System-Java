# Hospital-Management-System-Java
Greetings, this project is a Hospital Management System that is coded in Java.

Instructions to run the file:
1. Download the ZIP file above.
2. Install Apache Netbeans
3. Import the zip file. (File > Import Project)
4. Find the file, Login.java in AdminDashboard.
5. Right click Login.java and run the file.
